Manual Bolding

What we call "Manual Bolding" is that the user (you) have already decided exactly which nucleotides you would like to be which font weight. You may have decided this "by hand", or you may have adapted your computer programs to output to our input format. Here is the Font Weight documentation.

For this example, we have decided to set the font weight for each nucleotide in the Small Subunit based on identity. G's will be 900 (most bold) C's will be 700 (normal bold), A's will be 400 (normal), and U's will be thin (100). In addition, we will set a column called "DataDescription". In the first row, we write a description of the data. It can be as detailed as you like. The resName column is just there for our convenience. resName is not currently a recognized input name in this mode and will be ignored.

Example

resNumresNameFontWeightDataDescription
16S:1A400"This is essentially..."
16S:4U100
16S:1276G900
16S:1539C700
