Manual Editing

Color:

Recoloring an individual residue is done with the middle mouse button. Pushing down on the scroll wheel often counts as the middle button. My laptop's trackpad has two finger tap set to middle button. Your controls may vary. You can color with the left mouse button if one button mode is set to color. Alternatively, you can color with the left mouse button if you hold down Ctrl. This feature now supports layers. You will recolor the letter or circle of the selected layer.
