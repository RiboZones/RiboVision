Legend

JAT: Java Applet always on top problem. Annoying, but not critical.

NUD: No User-Generated Data Support yet. This will be fixed in the future by allowing the server to process this data. Our maybe allow it to be copy and pasted into the browser.

MV #: The maximum version for this browser is dependent on the OS. This is the highest version available for that OS.

GAP: OSX Lion and Mountain Lion come with 64-bit Java 7. Apple does not provide a 32-bit Java 7. Google does not provide a 64-bit Chrome. So your choices are to not use Google Chrome on Lion+, or to downgrade to Java 6. (not reccommended). Hopefully Google will make a 64-bit Chrome soon.
