Load a Molecule:

The first thing you need to do is “Select” a species and a ribosomal subunit from the Species and Subunit menu. RiboVision contains ribosome data for E. coli, T. thermophilus, H. marismortui, and S. cerevisiae species. For this example, we will use E. coli.Further in this menu you may choose either the large subunit or small subunit. We will choose the large subunit. We select “load data” to continue.

Upon selecting the map, the 2D panel will immediately load. We have different default views for each panel. You will see the secondary structure with black letters. Up on the NavLine is what we call the Rainbow. You can see residue sequence indices from 0 to about 3000 for each residue. The 3D view of the Ribosome loads in the Jmol panel on the right. Please be aware that the Jmol panel may take a minute or two to load. You can see that the default color scheme illustrates the six different domains of ribosomal RNA.

Load Data:

From here we can load data. To do this, click on one of the Data Sections in the Main menu. We have the “Structure Data”, “Alignment data”, “Protein Data”, and “Interaction data” sections. See descriptions of the data in the “Our Data” seciton of this manual. Also, check out this YouTube video.

Loading data from the Structure Data and Alignment Data sections is easy. Just select a dataset from the relevant menu. From here, the data shown in the 1D, 2D, and 3D panels are synchronized, unless you use advanced techniques.

In the Protein data section, you can chose to visualize the entire set of the proteins for a ribosomal subunit, or you can select a desired protein or subset of proteins from the list in the Protein data menu by checking the boxes. Upon selection of a set or subset of proteins, you can see that on the 2D panel, the colors of the residue letters have changed. Colored residues are residues that are in contact with a protein in the 3D structure; each residue color in the 2D structure corresponds to a protein of the same color now seen in the Jmol panel.

You can also graph 3D interaction data in the 2D panel, which are shown as lines between residues that are close in 3D space according to various criteria. We will demonstrate this feature by selecting “Base Pair interactions” from the “Interaction Type” menu. Some interaction types can be further characterized into subtypes, so we clear out the “Interaction Sub-Type” menu, and choose, for example, “cis Watson Crick” base pairs, abbreviated as “cWW”. You will see that lines appear between some residues. Each line connects two residues forming a Watson Crick base pair.
